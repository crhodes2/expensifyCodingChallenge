Running the PHP Proxy requires a local PHP server. It is actually what makes the website functional.
If running the program on a local drive, I HIGHLY encourage you to install XAMPP Apache Server, which
is what I used to develop this program.

Please follow the instructions on this link to see how to properly run this server.
https://www.edureka.co/blog/how-to-run-a-php-program-in-xampp/

Once you successfully run the server, the PHP files in this folder must be redirected in your localhost
